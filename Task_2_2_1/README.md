## Эмулятор пиццерии
В данной программе реализован эмулятор пицерии.
 
Структура классов пиццерии представлена на следующей диаграмме
![PizzeriaUML.png](PizzeriaUML.png)
Рис. 1. Представление пиццерии
На данной диаграмме представлена работа пиццерии. Конфигурация пиццерии загружается 
из json файла с помощью класса Config. Он содержит загрузчик и конфигурацию.
Класс Pizzeria это основной класс приложения. Он содержит очередь заказов, ссылку на склад,
список поваров и курьеров.
У курьера есть ссылка на склад а так же список заказов, которые он доставляет 
в текщий момент времени. Курьер доставляет несколько заказов сразу.
Пекарь содержит ссылку на заказ.
У заказа есть статус.
